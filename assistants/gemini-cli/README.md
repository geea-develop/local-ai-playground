# ♊ Gemini CLI

> [!NOTE]
> **Status: 🚧 In Progress** — This directory is a placeholder for the Gemini CLI integration. Setup and examples will be added here.

[Gemini CLI](https://github.com/google-gemini/gemini-cli) is Google's open-source AI agent for the terminal. It brings Gemini's capabilities directly into your command line, enabling code generation, file editing, shell command execution, and web search — all via natural language.

---

## Planned Setup

1. **Install Gemini CLI**:
   ```bash
   npm install -g @google/gemini-cli
   ```

2. **Authenticate**:
   ```bash
   gemini auth login
   ```

3. **Start a session**:
   ```bash
   gemini
   ```
