# AI Frameworks & SDKs

This directory contains experiments, proof-of-concepts, and implementations using various AI orchestration frameworks and software development kits.

## Purpose
The focus here is on **code-centric** orchestration. Unlike the `interfaces/` directory (which focuses on UIs), these projects are about building agentic behavior, tool use, and memory management through code abstractions.

## Contents

- **[langchain/](./langchain)**: Experiments and evaluations using the LangChain library (Python/JS).
- **[smolagents/](./smolagents)**: Lightweight agent construction using Hugging Face's `smolagents` framework.

## When to use these
Use these frameworks when you are building a custom application, a specialized agent, or a backend service that requires programmatic control over the LLM's logic and tool-calling capabilities.
