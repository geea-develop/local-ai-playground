# AI Assistants & Coding Utilities

This directory holds the configuration, research, and usage guides for pre-built AI assistants and specialized coding tools.

## Purpose
The projects here are designed to be **autonomous or semi-autonomous helpers** that assist with development workflows, coding tasks, and system management.

## Contents

- **[goose/](./goose)**: Configuration and documentation for the Goose open-source AI agent.
- **[claude-cli/](./claude-cli)**: Custom skills and plugins for the Anthropic Claude CLI.
- **[claude-lm-studio/](./claude-lm-studio)**: Integrations connecting Claude's capabilities with local LM Studio inference.

## When to use these
Leverage these assistants when you need an "expert-in-the-loop" to help write code, debug issues, or automate repetitive terminal-based tasks.
